# ⚖️ NyayaShield | AI-Powered Legal Assistant

NyayaShield is a professional legal and consumer complaint drafting assistant designed to bridge the gap between complex legal terminology and common citizens. It empowers users to articulate their grievances into professional, court-ready legal documents without needing expensive legal counsel for the initial draft.

🚀 **Live Demo:** [Click here to explore](https://project-rakshak-tr98.onrender.com)

## 🛠️ The Problem
In India, high legal complexity and a lack of awareness regarding rights (BNS/BSA) often prevent citizens from seeking justice. Most people struggle to translate their real-world problems into the formal, authoritative language required by judicial bodies.

## ✨ The Solution
NyayaShield leverages a custom-tuned "Hinglish" AI engine to convert informal, conversational user inputs into sophisticated, formal, and court-ready legal drafts.

### Key Features:
- **Hinglish Support:** Native understanding of natural Indian language patterns, allowing users to describe problems in the way they actually speak.
- **Professional Drafting:** Transforms raw input into authoritative legal complaints adhering to formal standards.
- **Product Growth Logic:** Integrated WhatsApp sharing mechanism to drive organic user acquisition and democratic access to legal aid.

## 🧠 Technical Implementation (The Logic)
- **Prompt Engineering:** Implemented a specialized system prompt to handle the transformation of conversational text to formal legal prose while maintaining the core facts of the grievance.
- **Low-Latency Inference:** Integrated **Groq API (Llama 3.3)** to ensure near-instant generation of drafts, providing a seamless user experience.
- **Input Sanitization:** Built a pipeline to handle messy, unstructured user input and map it to a structured legal format.

## 💻 Tech Stack
- **Backend:** Python (Flask)
- **AI Core:** Groq API (Llama 3.3)
- **Frontend:** JavaScript (ES6+), CSS3
- **Deployment:** Render
